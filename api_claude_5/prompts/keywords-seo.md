REGLAS ESTRICTAS:
✓ MÁXIMO 4 palabras por keyword (incluyendo artículos)
✓ NO usar preguntas completas (cómo, qué, dónde con verbos)
✓ SÍ usar sustantivos + modificadores
✓ Priorizar términos que la gente escribe en Google (keywords reales de búsqueda)
✓ Incluir variantes con ciudad si es negocio local
✓ Usar modificadores de intención: comprar, precio, mejor, barato, profesional, cerca

CRITERIOS DE SELECCIÓN:
✓ Intención de búsqueda clara (informacional, transaccional, navegacional)
✓ Oportunidades de bajo volumen + baja competencia (sweet spot)
✓ Keywords relacionadas con servicios/productos específicos del negocio
✓ Variaciones semánticas y sinónimos
✓ Keywords de tendencia actual (2024-2025)
✓ Términos de comparación si aplica (vs, mejor que, diferencia entre)
✓ Keywords geolocalizadas si el negocio es local

EVITAR:
✗ Keywords demasiado competidas sin modificadores
✗ Términos genéricos sin contexto
✗ Keywords sin volumen de búsqueda realista
✗ Repetir la misma keyword con ligeras variaciones
